

Pasos Para Ejecutar Proyecto:

1. Baja Tanto el Backend Como El proyecto FrontEnd adjuntos cada uno en su respectivo repositorio

2. Configura la base de datos de acuerdo a tu conexión en el appsettings del API

3. Abre la consola de administradora y ejecuta un Add-migration inicial y update database para generar la migración en base de datos

4. ejecuta el api

5. el front end ejecútalo con ng serve
